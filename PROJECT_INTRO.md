# AirSteady 飞影稳拍 - 项目协作计划

## 一句话介绍

给拍飞机/航模/打鸟的人，用 AI 自动跟拍 + 稳像他们的视频，让长焦手抖、主体乱飘的素材变得"像有云台一样好看"。

## 项目现状

### 已完成功能 ✅

- **视频加载**：支持 mp4/avi/mov/mkv 等格式，FFmpeg 硬件解码
- **代理视频**：自动生成低分辨率 proxy.mp4 加速处理
- **目标检测**：YOLO 分割模型 + ONNX Runtime DirectML GPU 加速
- **目标跟踪**：KLT 光流 + RANSAC 帧间位移估计
- **轨迹优化**：两步 IRLS 全局优化（速度/加速度平滑 + Huber 鲁棒核）
- **实时预览**：左右分屏对比（原始 vs 稳像）
- **视频导出**：按优化轨迹裁切输出，支持 AAC 音频复制/转码
- **参数调节**：平滑度、裁切比例、中心偏移

### 技术栈

| 领域 | 技术 |
|------|------|
| UI 框架 | Qt6 (Widgets) |
| 语言 | C++17 |
| 视频处理 | FFmpeg (硬解) |
| 推理后端 | ONNX Runtime + DirectML |
| 目标检测 | YOLO Segmentation |
| 特征跟踪 | OpenCV KLT 光流 |
| 轨迹优化 | IRLS + Huber 核 |
| 构建系统 | CMake + vcpkg |

### 代码结构

```
AirSteady/
├── exe/              # 程序入口
├── ui/               # Qt 界面 (MainWindow, VideoView, ControlPanel)
├── algorithm/        # 核心算法 (Processor, Tracker, Stabilizer...)
├── yolo/             # YOLO 推理 (ONNX Backend, Postprocess)
├── common/           # 通用工具
└── CMakeLists.txt
```

## 如何参与？🙋

### 参与方式

1. **提交 Issue**：bug 报告/功能建议
2. **认领任务**：在 Issue 下评论认领想做的功能
